# Customer Management System

A web application for managing customer details. This project consists of a frontend built with React and a backend API. The application allows users to add, view, and manage customer information.

## Features

- Add new customers
- View customer details
- Edit customer information
- Delete customers
- Responsive design for both web and mobile devices

## Technologies Used

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (assuming a typical setup)

## Usage

1. **Add a Customer** : Navigate to the `Add Customer` page to fill out and submit the form.
2. **View All Customers** : Navigate to the `View All Customers` page to see a list of all customers.
3. **Edit a Customer** : Click on the edit button next to a customer in the list to modify their details.
4. **Delete a Customer** : Click on the delete button next to a customer to remove them from the database.

## API Endpoints

* **GET** `/api/customers`: Fetch all customers
* **GET** `/api/customer/:id`: Fetch a specific customer by ID
* **POST** `/api/addCustomer`: Add a new customer
* **PUT** `/api/updateCustomer/:id`: Update an existing customer
* **DELETE** `/api/customer/:id`: Delete a customer
